# React Router Navigation

## Description
A simple React application demonstrating navigation between multiple views using React Router.

## What's Built
- Home Page
- About Page
- Contact Page

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/react-router-navigation.git
   cd react-router-navigation
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm run dev
   ```
4. Open your browser and go to `http://localhost:3000`
